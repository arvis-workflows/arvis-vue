# arvis-vue

> Search for Vue.js API references on vuejs.org

<img src="media/screenshot.png">

## Install

```
$ npm install -g arvis-vue
```

*Requires [Node.js](https://nodejs.org) 4+*


## Usage

In arvis, type `vue`, <kbd>Enter</kbd>, and your query.
